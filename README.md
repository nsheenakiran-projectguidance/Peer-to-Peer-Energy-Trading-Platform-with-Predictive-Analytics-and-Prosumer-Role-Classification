# ProNet Energy Trading and Forecasting System

## Overview
ProNet Energy is a smart web-based platform developed to predict household energy consumption and solar energy production using Machine Learning. The system also supports Peer-to-Peer (P2P) energy trading between producers and consumers.

This project aims to improve energy management in smart grids by enabling efficient forecasting and energy sharing.

---

## Features

- User Registration and Login
- Producer and Consumer Dashboards
- Energy Consumption Prediction
- Solar Energy Production Forecasting
- Peer-to-Peer Energy Trading
- CSV Dataset Upload
- Trading History Management
- Prediction Result Visualization
- Admin Panel
- Responsive Web Interface
